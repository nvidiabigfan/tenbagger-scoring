# 텐배거스코어링 -- PRD (Product Requirements Document)

> 생성일: 2026-05-25
> 생성 도구: Show Me The PRD

---

## 1. 제품 개요

### 한 줄 요약
미국주식 종목 1개를 입력하면 Reddit·ETF편입·애널리스트·유튜브 4개 채널을 종합 분석해 "텐배거(10배 상승) 후보 가능성"을 0~100점 스코어로 보여주는 무료 웹 대시보드.

### 해결하는 문제
- 개인투자자는 텐배거 후보를 발굴할 때 **여러 채널을 일일이 돌며** 리테일 관심도(Reddit/유튜브), 패시브 자금 유입(ETF 편입), 펀더멘털(애널리스트·PER)을 따로 확인해야 함
- 각 채널의 시그널을 **하나의 점수로 종합**해주는 무료 서비스가 없음
- 기존 스크리너(Koyfin·TradingView 등)는 펀더멘털·기술적 지표 위주라 **소셜·내러티브 시그널이 약함**

### 핵심 가치
- **다채널 종합 스코어**: 펀더멘털 + 소셜 + 수급 + 내러티브를 단일 점수화
- **텐배거 특화**: 단순 매수/매도가 아닌 "10배 후보" 관점 (초기 시그널 가중)
- **근거 리포트**: 점수만 던지지 않고 모듈별 근거(Reddit 핫 게시물, ETF 편입 변화, 애널리스트 코멘트, 유튜브 멘션) 함께 제공
- **무료 공개**: 인지도 확보 목적, 가입·구독 없이 일부 기능 사용 가능

## 2. 사용자

### 주요 사용자
- **누구**: 미국주식에 관심 있는 한국 개인투자자 (특히 성장주·테마주 발굴 관심층)
- **상황**: 뉴스·유튜브·커뮤니티에서 종목을 듣고 "이거 진짜 갈까?" 1차 스크리닝이 필요할 때
- **목표**: 종목 하나의 "다채널 신호 강도"를 5분 안에 파악

### 사용자 시나리오
1. 사용자가 유튜브에서 "AVGO가 다음 NVDA 후보다"라는 영상을 보고
2. 텐배거스코어링 사이트에 접속해 "AVGO" 검색
3. 5분 안에 통합 스코어 + 4개 모듈 근거 리포트를 받음
4. 마음에 들면 워치리스트 등록 → 스코어 변화 시 이메일 알림 수신
5. 매일 갱신되는 "스코어 상위 100" 랭킹 보드도 둘러봄

## 4. 사용자 흐름 (User Flow)

### 핵심 흐름 (On-demand 분석)
```
사이트 접속 -> 종목 검색 -> 분석 시작(5분 대기) -> 통합 리포트 페이지 -> [선택] 워치리스트 추가
```

### 핵심 흐름 (워치리스트 모니터링)
```
이메일 로그인 -> 워치리스트 종목 선택 -> 일별 스코어 변화 확인 -> 임계치 초과 시 이메일 자동 수신
```

### 핵심 흐름 (랭킹보드 탐색)
```
사이트 접속 -> 랭킹보드 페이지 -> 상위 100 종목 스캔 -> 관심 종목 클릭 -> 상세 리포트
```

### 상세 흐름 (On-demand)
1. **종목 검색**: 티커 또는 회사명 입력, 자동완성으로 정확한 종목 선택
2. **분석 큐 등록**: 백엔드 4개 모듈을 병렬 호출 (Reddit·ETF·애널리스트·유튜브)
3. **진행률 표시**: 모듈별 진행 상황 실시간 표시 (1~5분 소요)
4. **통합 리포트 렌더링**: 총점 + 시그널(buy/hold/sell) + 모듈별 점수·근거
5. **액션**: 워치리스트 추가 / 리포트 공유 / 다른 종목 비교 (P2)

## 5. 성공 기준

### Phase 1 (MVP) 성공 기준
- [ ] 종목 입력부터 통합 리포트 표시까지 **5분 이내** 완료 (95th percentile)
- [ ] 4개 모듈 각각 **신뢰도 70% 이상**의 분석 결과 출력 (백테스트 기반)
- [ ] **월 활성 사용자(MAU) 100명** 달성 (출시 후 3개월 내)
- [ ] **워치리스트 등록 종목당 일 평균 1회** 자동 재분석 안정 운영
- [ ] 인지도 지표: 한국어 투자 커뮤니티(클리앙·디시 주식갤·블라인드) **자발적 언급 10건 이상**
- [ ] X/트위터 리포트 공유 클릭 → 사이트 유입 **CTR 5% 이상** (P2 이후)

### 비즈니스 성공 기준 (추후 상품화 검토용)
- [ ] 워치리스트 사용자 비율 30% 이상 → 유료 전환 가능성 검증
- [ ] 사용자 1인당 평균 분석 종목 수 5개 이상

## 6. 안 만드는 것 (Out of Scope)

> 이 목록에 있는 건 Phase 1에서 만들지 않습니다.
> AI에게 코드를 시킬 때 이 목록을 함께 공유하세요.

- **한국주식 분석** -- 이유: 텐배거 본질이 미국 시장 컨셉. 한국은 별도 프로젝트(09_텔레그램신호감지·14_미국주식투자어시스턴트 등)에 위임
- **실시간 시세·차트** -- 이유: TradingView·Yahoo가 압도적. 본 서비스는 "스코어링"에 집중
- **매수·매도 주문 연동** -- 이유: 증권사 API 연동 → 규제·책임 리스크. 8번·14번 프로젝트 영역
- **AI 챗봇·자유 질의** -- 이유: 정형화된 스코어링이 핵심 가치. 챗봇은 산만함 유발
- **모바일 네이티브 앱** -- 이유: PWA로 충분. iOS·안드로이드 빌드는 P2 이후 검토
- **유료 결제·구독** -- 이유: 인지도 확보 목적 무료 공개. 상품화는 Phase 3 검증 후 결정
- **공시 한글 번역·해석** -- 이유: SEC EDGAR 원문 링크 제공으로 충분. 번역은 LLM 비용 부담
- **포트폴리오 관리** -- 이유: 워치리스트와 다름. 보유 종목 관리는 별도 서비스 영역
