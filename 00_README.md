# 21_텐배거스코어링

## 개요
미국주식 종목 1개를 입력하면 **Reddit·ETF편입·애널리스트·유튜브 4개 채널을 종합 분석**해 "텐배거(10배 상승) 후보 가능성"을 0~100점 스코어 + 근거 리포트로 보여주는 무료 웹 대시보드. 기존 여러 분석 프로젝트의 통합 허브.

## 핵심 가치
- **다채널 종합 스코어**: 펀더멘털 + 소셜 + 수급 + 내러티브 단일 점수화
- **텐배거 특화**: 초기 시그널 가중 (단순 매수/매도가 아님)
- **모듈 확장 구조**: 허브-스포크. 신규 모듈(SEC·인사이더·어닝스콜·공매도) 추가 시 인터페이스만 준수
- **무료 공개**: 인지도 확보 목적
